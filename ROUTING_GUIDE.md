## 🚀 Routing System

### How It Works
The app uses **dynamic route loading** based on folder names:

```javascript
// Automatic detection - no manual routes needed!
const htmlModules = import.meta.glob("../*/code.html", {
  query: "?raw",
  import: "default"
});

// Routes accessible via hash URL:
// http://localhost:5173/#/home
// http://localhost:5173/#/admin_overview
// http://localhost:5173/#/baytmiftah_home
// etc.
```

### Current URL Structure
- **Pattern**: `http://localhost:5173/#/FOLDER_NAME`
- **All 125 screens are auto-discovered** and routable
- **Search functionality** built-in with `switcher-search`
- **Screen counter** shows "125 routes"
